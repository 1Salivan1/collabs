| ------------ | ------------------------------------------------------------------- | ------------ |

| creator_id   | Повинен бути рядком                                                 | ✅           |

| discord      | Рядок, не обов’язкове                                               | ❌           |
| linkedin     | Рядок, не обов’язкове                                               | ❌           |
